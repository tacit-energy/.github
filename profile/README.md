# Tacit

Operational memory for energy systems.

[Repository](https://github.com/tacit-energy/tacit) · [Open Demo](https://hackathon.re4vive.com/) · [See How It Works](https://tacit.melde.net/)

Tacit turns thousands of messy sensor streams into clear signals. We capture human judgment instead of replacing it, bridging the gap between statistical anomaly detection and your team's institutional knowledge.


## Why Tacit?

### Don’t outsource the thinking. Empower the human.

Tacit turns noisy plant telemetry into clear operational signals. We capture human judgment instead of replacing it, closing the gap between raw anomaly detection and the institutional knowledge your team relies on.

## The dilemma

Automation without context is just noise.

Complex energy systems generate billions in wasted energy. When a pure AI flags an anomaly and dictates a shutdown, operators reject it. It’s a black box. Meanwhile, senior engineers retire and the hard-won expertise required to run these plants walks out the door.

> "Data-purists want math, but operators need human context."

## The solution

Tacit is a topology-aware copilot that speaks two languages. Every insight is labeled with confidence so operators know whether a claim is hard data or learned judgment.

### Hard Data
- Statistical anomaly detection across complex network topology
- Uses conservation laws and deterministic facts

### Operator Memory
- Learns from operator overrides, acceptances, and dismissals
- Captures the "why" behind decisions and stores institutional knowledge

## Product philosophy

### Human-in-the-loop
- Listens to and learns from the operator
- Captures reasoning with every decision

### Radical transparency
- Always shows whether a claim is hard data or inferred
- No false certainty or generic assertions

### Clarity in complexity
- Reduces cognitive load
- Removes chat noise and non-essential telemetry

### Cost-efficient compute
- Session memory caching prevents unnecessary recomputation
- Keeps live analysis efficient

## Under the hood

Tacit integrates with industrial SCADA and BACnet controllers, displaying real-time telemetry alongside compounding human context.

### Live Topology Graph
- Shows network structure clearly
- Highlights only the anomalies that matter

### Exploration Table
- Filterable raw IoT streams for technical verification
- Fast, millisecond query response

### Decision Log
- Auditable memory of overrides, dismissals, and acceptances
- Preserves your plant’s tacit knowledge over time
